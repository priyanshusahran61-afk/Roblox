# Bloxcraft

A web-based Roblox clone built with Next.js, React Three Fiber, and Rapier physics.

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Controls

- WASD / arrow keys: move
- Space: jump
- Drag mouse: rotate camera
- Scroll: zoom
